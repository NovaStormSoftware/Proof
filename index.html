Proof by NovaStormSoftware

Fast Document Previewing

(This isn't totally complete yet, but bear with us...) Proof is a fast document preview tool that is specifically designed for use with exporters. If you've used an export tool that doesn't have a built-in preview, then you know how frustrating it is to go back and forth and constantly open, view, and delete bad versions until you have it right. Proof streamlines that process by providing a single application to preview all the files in a folder. When you're satisfied, you can use the Accept button to move those files to their final destination folder.
